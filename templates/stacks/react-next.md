## FRONTEND EXPERTISE (REACT/NEXT.JS)
- **Component Style:** STRICTLY use functional components and React Hooks. ABSOLUTELY NO class components.
- **Styling:** Prefer Tailwind CSS utility classes for styling.
- **Next.js Architecture:** Default to App Router and Server Components. Only use `'use client'` directive when browser APIs or interactivity (hooks like useState/useEffect) are explicitly required.
- **State Management:** Keep state local whenever possible. Avoid overusing global state stores unless passing props becomes deeply nested.
